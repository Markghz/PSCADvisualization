# PSCAD See

The program is designed to read and show data from PSCAD's simulation result files.

#### How to use

- Run PSCADSee.jar
- Click the *File* menu, import *.inf* file and *.out* file (note the order).

#### Dependencies

- jcommon-1.0.23
- jfreechart-1.0.19
- jtransforms-3.1
- commins-maths3-3.6.1
- tablelayout
